# 55. 跳跃游戏

不看答案真的想不到用覆盖长度来做

```c++
class Solution {
public:
    bool canJump(vector<int>& nums) {
        if (nums.size() == 1) return true;
        int cover = 0;
        for (int i = 0; i <= cover; i++) {
            cover = max(cover, i + nums[i]);
            if (cover >= nums.size() - 1) return true;
        } 
        return false;
    }
};
```
