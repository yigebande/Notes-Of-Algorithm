# 135. 分发糖果

完全没有思路，刚开始以为是要从得分最小的孩子开始分两边找，但是提交之后有测试案例无法通过。

实际上出现多个最小得分的孩子的时候，情况就变得很棘手。

最后是看了文章讲解做的。

```c++
class Solution {
public:
    int candy(vector<int>& ratings) {
        vector<int> first(ratings.size(), 1);
        vector<int> second(ratings.size(), 1);
        for (int i = 1; i < ratings.size(); i++) {
            if (ratings[i] > ratings[i - 1]) {
                first[i] = first[i - 1] + 1;
            } else {
                first[i] = 1;
            }
        }
        for (int i = ratings.size() - 2; i >= 0; i--) {
            if (ratings[i] > ratings[i + 1]) {
                second[i] += second[i + 1];
            } else {
                second[i] = 1;
            }
        }
        int sum = 0;
        for (int i = 0; i < ratings.size(); i++) {
            sum += max(first[i], second[i]);
        }
        return sum;
    }
};
```
