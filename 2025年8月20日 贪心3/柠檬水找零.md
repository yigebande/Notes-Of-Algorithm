# 860. 柠檬水找零

刚开始我以为是全部收完钱再找零，费了好大功夫写，提交之后才发现是按顺序一边收钱一边找零。

这样的话问题就很简单了。

```c++
class Solution {
public:
    bool lemonadeChange(vector<int>& bills) {
        int five = 0;
        int ten = 0;
        for (int i = 0; i < bills.size(); i++) {
            if (bills[i] == 5) {
                five++;
            } else if (bills[i] == 10) {
                ten++;
                five--;
            } else {
                if (five >= 1 && ten >= 1) {
                    five--;
                    ten--;
                } else if (five >= 3) {
                    five -= 3;
                } else {
                    return false;
                }
            }
            if (five < 0 || ten < 0) return false;
        }
        return true;
    }
};
```

纯`if else`局
