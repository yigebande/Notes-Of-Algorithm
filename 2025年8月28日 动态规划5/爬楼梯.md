# 卡码网57. 爬楼梯
一道标准的完全背包求排列数题目。

背包容量为`m`，每次可以上`[1, n]`阶楼梯，且`n < m`。

这里的 **可上楼梯阶数** 就相当于物品大小。

```c++
#include<iostream>
#include<vector>
using namespace std;

int main() {
    int m, n;
    cin >> m >> n;
    vector<int> dp(m + 1, 0);
    dp[0] = 1;
    for (int i = 0; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (i >= j) {
                dp[i] += dp[i - j];
            }
        }
    }
    cout << dp[m] << endl;
    return 0;
}
```
