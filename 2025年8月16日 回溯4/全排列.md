# 46. 全排列

比较简单的一道题，思路如下：

假设只有三个元素，那么第一个位置可以有三种选择，第二个位置有两种选择，第三个位置只有一种选择。每个位置的可供选择的元素不能与前面已经选择的元素相同。

```c++
class Solution {
public:
    vector<vector<int>> res;
    vector<int> path;
    bool umap[7] = {0};

    void backTracking(vector<int>& nums) {
        if (path.size() == nums.size()) {
            res.push_back(path);
            return;
        }
        for (int i = 0; i < nums.size(); i++) {
            if (umap[i] == 1) {
                continue;
            }
            path.push_back(nums[i]);
            umap[i] = 1;
            backTracking(nums);
            path.pop_back();
            umap[i] = 0;
        }
    }

    vector<vector<int>> permute(vector<int>& nums) {
        backTracking(nums);
        return res;
    }
};
```
