# 卡码网 54. 替换数字

很简单的一道题

```c++
#include<iostream>
#include<string>
using namespace std;

int main() {
    string s;
    string new_s = "";
    cin >> s;
    for (int i = 0; i < s.length(); i++) {
        if (s[i] >= '0' && s[i] <= '9') {
            new_s += "number";
        } else {
            new_s += s[i];
        }
    }
    cout << new_s << endl;
    return 0;
}
```
